# exposiciones-backend

API REST para el sistema de evaluacion de exposiciones academicas.

## Requisitos
- Node.js 20 LTS
- Cuenta en Supabase
- Cuenta en Vercel

## Instalacion
1. Clonar el repositorio
2. Copiar .env.example a .env y completar las variables
3. npm install
4. npm run dev

## Ramas
- main: produccion
- develop: integracion
- feature/*: desarrollo por modulo

## Convencion de commits
feat(modulo): descripcion
fix(modulo): descripcion
chore(modulo): descripcion
docs(modulo): descripcion
